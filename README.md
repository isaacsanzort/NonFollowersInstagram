# VER NON-FOLLOWERS INSTA
Esta aplicación es para parsear y obtener una lista de los no seguidores de Instagram, que se obtienen de los datos que pides a Instagram
## COMO OBTENER LOS DATOS
Desde PC: Acceder a Instagram > Configuración y privacidad > Centros de cuenta (https://accountscenter.instagram.com/info_and_permissions/) > Descargar tu información > Solicitar descarga  > Seleccionar información > Seguidores y seguidos > Intervalo de fechas (Desde el principio) + **Formato JSON**

Una vez solicitada la información, te llegará un correo con los archivos
